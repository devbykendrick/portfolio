    Instagram Python Bot

    Python, Miniconda, Crontab

    This is a Python-based Instagram bot that automates the process of posting inspirational quotes on your Instagram account. The bot pulls random images and inspirational quotes from API Ninjas platform and overlays the quotes onto the images before posting them to your Instagram feed. The automation is handled by Miniconda and Crontab where the account posts every 12 hours.

    Sushi Restaurant

    React, React Router

    This dynamic website uses React and React Router to simulate the online operations of a sushi restaurant. Users can easily add food items to their cart, view the cart contents, and track associated costs.     IP Address Map

    JavaScript (Axios), HTML, CSS

    This IP address search tool uses JavaScript and Axios, integrating two APIs for functionality. The tool incorporates LeafletJS to display the map and utilizes IPify for IP address searches. Additionally, Axios is employed to connect with backend APIs, enabling seamless communication between the frontend and backend components of the application.
